# Comfyui-LinuxMintinstaller
Comfyui-LinuxMintinstaller
ак пользоваться программой после установки

Запусти установщик
Кликни по ярлыку ComfyUI Installer в меню «Графика»
Или в терминале набери:Bashcomfyui-installer

Что ты увидишь в окне
Заголовок: «ComfyUI Installer»
Поле «Папка установки» (по умолчанию ~/ComfyUI-Desktop)
Кнопка «Выбрать» — если хочешь установить в другую папку (например, на внешний диск)
Выбор профиля видеокарты (Auto, CPU, low-vram и т.д.)
Кнопки:
Тестирование пакетов — проверяет, всё ли готово
Установить / Обновить — основная кнопка
Прервать — если что-то пошло не так
Открыть ComfyUI — после установки


Процесс установки
Нажми Установить / Обновить
Программа проверит пакеты → если чего-то нет — предложит установить (в отдельном терминале)
Скачает ComfyUI с GitHub
Создаст виртуальное окружение (venv)
Установит PyTorch (CPU-версия или CUDA, в зависимости от выбора)
Установит все зависимости из requirements.txt
Создаст ярлык в меню «Графика»

После успешной установкиТы увидишь сообщение:textУстановка завершена!

Путь к ComfyUI: /путь/который/ты/выбрал/ComfyUI-Desktop/ComfyUI

Ярлык: /home/твой_логин/.local/share/applications/comfyui.desktop

Как запустить:
1. Меню → Графика → ComfyUI
2. Дождитесь строки "To see the GUI go to: http://127.0.0.1:8188"
3. Браузер откроется автоматически
Запуск ComfyUI
Найди в меню «Графика» → ComfyUI (не Installer, а просто ComfyUI)
Кликни → откроется терминал + браузер
Дождись строки в терминале:textTo see the GUI go to: http://127.0.0.1:8188
Вкладка в браузере откроется автоматически

Обновление ComfyUI
Запусти ComfyUI Installer снова
Нажми Обновить ComfyUI (или ту же кнопку «Установить / Обновить»)
Программа обновит репозиторий и зависимости

Удаление пакета (если захочешь убрать установщик)Bashsudo apt remove comfyui-installerЭто удалит только сам установщик. ComfyUI останется в выбранной папке.

Полезные советы

Если после запуска ComfyUI браузер не открылся — просто открой в браузере:
http://127.0.0.1:8188
Если терминал закрылся — просто снова запусти ярлык ComfyUI
Хочешь переустановить в другую папку — удали старую папку и запусти установщик заново
